VisionCore

VisionCore is a production-ready Python image recognition system with a full web frontend. It supports image classification, object detection, and batch processing across TensorFlow, PyTorch, and OpenCV backends.

Quick start: See the How to Run section to be up in 5 minutes.
// installation

Installation

Requires Python 3.9+. Open Command Prompt (cmd) in your project folder:

# Install all required packages
pip install flask flask-cors Pillow numpy opencv-python

# Optional: for real ML inference
pip install tensorflow          # TensorFlow backend
pip install torch torchvision   # PyTorch + YOLO backend
Note: Without TensorFlow/PyTorch the app runs on mock/demo data — perfect for testing the UI.
// how to run

How to Run

# Step 1 — Open Command Prompt (cmd), go to project folder
cd D:\visioncore

# Step 2 — Install packages (first time only)
pip install flask flask-cors Pillow numpy opencv-python

# Step 3 — Start the server
python api_server.py

# Step 4 — Open browser
http://localhost:5000
Next time: Just run python api_server.py and open http://localhost:5000
// python api

ImageRecognizer

Multi-backend image classification using pre-trained ImageNet models.

from src.image_recognizer import ImageRecognizer

r = ImageRecognizer(backend='tensorflow', top_k=5)
preds = r.predict('photo.jpg')

for p in preds:
    print(f"#{p['rank']} {p['label']} — {p['confidence']:.2%}")
ParameterOptionsDefault
backendtensorflow · torch · opencvtensorflow
top_kAny integer5

ObjectDetector

from src.object_detector import ObjectDetector

d    = ObjectDetector(backend='yolo', confidence_threshold=0.5)
dets = d.detect('photo.jpg')
d.annotate_image('photo.jpg', 'annotated.jpg')

Preprocessing

from utils.preprocessing import load_image, resize_image, augment_image

img = load_image('photo.jpg')

img = augment_image(img, flip_h=True, brightness=1.2)

CLI Reference

# Classify a single image
python main.py classify --image photo.jpg --backend tensorflow --top-k 5

# Detect objects
python main.py detect --image photo.jpg --backend yolo --annotate

# Batch classify a folder
python main.py batch --folder data/ --output results.json

# Show image info
python main.py info --image photo.jpg
